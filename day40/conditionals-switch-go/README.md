# ⚡ Day 40: Conditionals (switch) — "Режимы дня Гоши"

**Дата**: 13 декабря 2025 г.
**Тема**: Условные операторы `switch` в Golang
**Легенда**: Программа, написанная за полчаса до субботнего обеда, пока мама на кухне варит рис. Вдохновлена днём, когда мир кажется холодным, а друг работает удалённо из дома. Но Гоша читает книгу по Паскалю в метро, подставляя в уме синтаксис Go.

## ❄️ Как это работает
Программа моделирует твой сегодняшний день через конструкцию `switch`:
- Определяет часть дня по времени
- Анализирует настроение на основе недосыпа
- Выбирает "режим работы" в зависимости от обстоятельств
- Предлагает интерактивный выбор следующего шага

### Ключевые конструкции Go:
```go
// Switch без условия (как серия if-else)
switch {
case условие1:
    // код
case условие2:
    // код
default:
    // код
}

// Switch с выражением
switch choice {
case "1":
    // действие
case "2", "3":
    // общее действие
default:
    // действие по умолчанию
}
