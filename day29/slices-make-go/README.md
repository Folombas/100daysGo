# Гоша - Программист-Перерожденный (Slices: make())

## Описание

Эта программа представляет собой историю Гоши, бывшего рэп-артиста и клубного тусовщика, который решил изменить свою жизнь и стать Go-разработчиком. Программа демонстрирует его прогресс в изучении языка Go, особенно темы "Slices: make()".

## Запуск

Для запуска программы выполните команду:
```bash
go run main.go
