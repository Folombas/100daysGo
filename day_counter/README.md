# 🐹 Гофер-Грызун: Трекер Фокуса на Go

> *«Распыление создаёт дилетантов. Фокус создаёт мастеров. Я выбираю фокус.»*

## 🎯 Суть проекта
`day_counter` — это терминальный трекер прогресса в изучении Go. Он считает дни марафонов 100daysGo и Go365, помогая оставаться на пути и избегать распыления.

**Философия:** Фокусировка на изучении Go → Прогресс каждый день.

## ✨ Особенности

### 📊 Отслеживание прогресса
- **Два параллельных челленджа** — 100daysGo и Go365
- **Точный подсчёт дней** — правильный расчёт с первого дня
- **Визуальные прогресс-бары** — наглядное отображение прогресса
- **Уровни и фокус** — рост мастерства с каждым днём

### 🧠 Фокус на изучении
- **Ежедневные темы** — структурированное изучение Go
- **Конкретные задачи** — 5 практических заданий на день
- **Оценка понимания** — отслеживание прогресса в темах
- **Отказ от троллинга** — экономия энергии для кода

## 🚀 Установка

```bash
# Клонируйте репозиторий
cd day_counter

# Запустите программу
go run main.go
```

## 🧠 Для кого этот трекер

1. **Начинающие гоферы** — кто делает первые шаги в Go
2. **Участники марафонов** — 100daysGo и Go365
3. **Фокусирующиеся разработчики** — кто хочет системного прогресса
4. **Избегающие распыления** — кто экономит энергию для важного

## 📜 Принципы использования

1. **Запускай ежедневно** — отслеживай прогресс по челленджам
2. **Фокусируйся на теме дня** — выполняй практические задания
3. **Игнорируй троллей** — не трать энергию на срачи в сети
4. **Пиши код каждый день** — даже если это всего 10 строк

## ⚠️ Важное правило

**Энергия для кода, а не для троллей.**
Каждая сэкономленная на срачах с троллями минута — это дополнительная строка кода, дополнительный изученный концепт, дополнительный шаг к мастерству.

## 🎯 Как получить максимум пользы

1. **Утром** — запусти трекер, посмотри тему дня
2. **В течение дня** — выполняй задачи по теме, пиши код
3. **При столкновении с троллями** — вспоминай: "Энергия для Go, не для них"
4. **Вечером** — гордись выполненными задачами, готовься к следующему дню
