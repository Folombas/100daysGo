# Variables & Constants: Основа стабильности в Go и жизни

## О модуле
Финальный день 100-дневного марафона! Этот модуль демонстрирует работу с переменными и константами в Go через метафору личной трансформации. Гоша учится отличать изменяемое (переменные) от неизменного (константы) в коде и жизни.

## Что внутри
- `main.go`: Демонстрация переменных и констант в контексте трансформации жизни
- Примеры разных типов переменных (int, float, string, bool)
- Глобальные и локальные переменные
- Константы как неизменные цели

## Как использовать
```bash
# Запуск программы
go run main.go

# Компиляция
go build

# Проверка типов
go vet
