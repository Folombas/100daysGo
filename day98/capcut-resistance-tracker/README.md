# Day 98: Concurrency Patterns — Pipeline in Golang

## Название модуля: CapCut Resistance Tracker (Трекер замещения видео-монтажа в CapCut на обучение языку Go и Go-стеку)

## Описание
Мини-программа, демонстрирующая паттерн "Конвейер" (Pipeline) в Go. Она подсчитывает, сколько дней прошло с момента принятия решения удалить отвлекающий софт (CapCut) и сфокусироваться на изучении Go. Программа — метафора силы воли, реализованная через горутины и каналы.

## Как работает конвейер
1.  `generateDaysSince()` — **Генератор**. Горутина, которая отправляет в канал все даты, начиная с 18.01.2026.
2.  `filterAndCount(<-chan time.Time)` — **Фильтр и Счётчик**. Читает канал, подсчитывая дни до текущей даты.
3.  `formatMessage(int)` — **Форматировщик**. Готовит полное сообщение с легендой.
4.  `printMotivationalPhrase(int)` — **Мотиватор**. Финальный этап, выдающий praise и мотивацию.

Это классический **pipeline**, где данные текут от одной стадии к другой через каналы.
