# Гоша - Программист-Перерожденный (Strings)

## Описание

Эта программа представляет собой 32-й день челленджа 100DaysGo, где Гоша, бывший рэп-артист и курьер, продолжает свой путь к становлению Go-разработчиком. Сегодняшняя тема — работа со строками в языке Go.

## Особенности

- Отслеживание прогресса Гоши на 32-й день челленджа
- Детальное описание сегодняшних трудностей с тяжелой посылкой
- Практические примеры работы со строками в Go:
  - Конкатенация
  - Поиск подстрок
  - Замена текста
  - Преобразование регистра
  - Работа с Unicode
- Мотивационные сообщения, связанные с темой строк
- Жизненные уроки, параллельные программным концепциям
- Интерактивная демонстрация трансформации "курьер" → "GO-РАЗРАБОТЧИК"

## Запуск

Для запуска программы выполните команду:
