# Generics in Go — Универсальные инструменты

## Описание модуля
Этот учебный модуль создан для 66-го дня челленджа **100daysGo: Reboot** и посвящён изучению **обобщённого программирования (Generics)** в Go. Generics позволяют писать гибкие, типобезопасные функции и структуры, работающие с различными типами данных, избегая дублирования кода или потери производительности.

## Философия дня
*За окном метель, ограничивающая пространство. В кармане — проездной с истекающим сроком. Но код, написанный с использованием Generics, — это "единый цифровой билет" к свободе. Он ломает старые ограничения, позволяя одной функции или структуре работать со множеством типов. Так и мы учимся создавать универсальные решения, которые делают нас сильнее и независимее.*

## Содержание
- `main.go` — демонстрационная программа с примерами:
  1. Универсальная функция `PrintSlice[T any]`, печатающая слайсы любого типа.
  2. Универсальная структура `Container[T any]`, хранящая значение любого типа.

## Как запустить
1. Убедитесь, что у вас установлен Go (версия 1.18 или выше).
2. В директории `generics-go` выполните:
   ```bash
   go run main.go
Вы увидите вывод, демонстрирующий работу дженериков со слайсами int, float64 и string, а также с обобщённой структурой.

Концепция
Generics в Go вводят параметры типа, обозначаемые в квадратных скобках (например, [T any]). Это позволяет:

Избежать дублирования кода для каждого типа.

Сохранить безопасность типов (в отличие от interface{}).

Повысить читаемость и поддерживаемость кода.
